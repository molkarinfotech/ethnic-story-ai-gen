# API Route Groups

Planned route groups for the backend:

- `/auth` - registration, login, session
- `/products` - product listing, detail, search
- `/collections` - category browsing
- `/cart` - add, remove, update cart
- `/checkout` - checkout session creation
- `/orders` - order history, order detail
- `/payments` - Stripe/Razorpay webhook handlers
- `/admin/products` - admin CRUD for products
